1) Открыть терминал в папке проекта
2) Инициализировать git в проекте командой (git init)
3) Добавить в .gitignore папку node_modules если она есть
4) Добавить все файлы проекта в git командой (git add .)
5) Сделать коммит командой (git commit -m "message")
6) Подключить удаленный репозиторий командой (git remote add origin git@example.com:example/test.git)
7) Загрузить проект на удаленный репозиторий командой (git push origin master)
